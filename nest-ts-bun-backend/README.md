# NestJS TypeScript Bun Backend Template

Production-grade NestJS API starter with TypeScript and Bun.

## Quick Start
```bash
bun install
cp .env.example .env
bun run dev
```

Server default URL: `http://localhost:3000`

## Core Scripts
- `bun run dev`
- `bun run start`
- `bun run typecheck`
- `bun run lint`
- `bun run test`

## Core Endpoints
- `GET /health`
- `GET /ready`
- `GET /metrics`
- `GET /api/v1/ping`
- `GET /openapi.json`
- `GET /docs`
